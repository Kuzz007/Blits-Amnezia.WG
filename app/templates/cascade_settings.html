Отдельный UDP-каскад на другой Amnezia-сервер

{% if settings.enabled == '1' %} Включен {% else %} Отключен {% endif %}
Обычная Amnezia 2.0 продолжает жить на порту {{ settings.main_v2_port }}, Legacy на {{ settings.main_legacy_port }}. Каскад должен использовать отдельные входящие порты, например 54912/54913, чтобы не перехватывать текущих клиентов. Для каскадного ключа Endpoint должен быть IP этой панели и порт каскада, а сам ключ/peer должен соответствовать целевому Amnezia-серверу.

Создание клиентов на целевой панели

Нужно для кнопки “Новый клиент → Каскад”: peer создается на целевом сервере, а ключ выдается с Endpoint этой панели.
