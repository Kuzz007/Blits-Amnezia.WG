Современная, быстрая и безопасная веб-панель управления VPN на базе **AmneziaWG** (форк WireGuard с защитой от блокировок DPI). Установка в одну команду, поддержка двух поколений клиентов, автовыпуск SSL и мощная серверная утилита администрирования.

---

## ✨ Основные преимущества

* **🚀 Развертывание за 70 секунд**: Полностью автоматический установщик ставит Docker, компилирует модуль ядра AmneziaWG, настраивает системный фаервол UFW и запускает контейнеры.
* **📱 Две версии AmneziaWG под любые клиенты**:
  * **Amnezia 2.0**: Полный профиль со случайными заголовками обфускации (`S1/S2/H1/H2/H3/H4`).
  * **Amnezia 1 / Legacy**: Облегченный профиль без параметров `S3/S4` для совместимости со старыми клиентами.
* **🔁 Динамическая серия QR-кодов**: Умное отображение QR-кодов частями с автоматической сменой слайдов каждые 2 секунды (упрощает импорт больших конфигов на мобильных устройствах).
* **🌐 Умный Split Tunneling (Маршрутизация)**: Раздельное туннелирование трафика по сетям, IP-адресам и доменным именам с готовыми пресетами (*Telegram, Discord, YouTube, OpenAI, Cloudflare*).
* **⚡ Консольный центр управления `blits`**: Полноценный CLI-интерфейс на сервере с графическим дашбордом системных ресурсов (CPU, RAM, Disk, Uptime) и встроенным ASCII QR-генератором.
* **🔒 Комплексная безопасность**:
  * Секретный префикс веб-пути входа (защита от сканирования роботами).
  * Встроенная защита административной панели от перебора паролей (Brute-Force Rate Limiting).
  * Логирование всех действий администратора (Audit Events log).
  * Автоматический выпуск и автопродление HTTPS-сертификатов Let's Encrypt для IP или домена.

## 🛠️ Быстрый старт

> [!IMPORTANT]
> Для корректной установки требуются права суперпользователя `root`. Поддерживаются операционные системы семейства Debian/Ubuntu (рекомендуется Ubuntu 22.04 LTS и новее).

Выполните команду на сервере для запуска автоматической интерактивной установки:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/kuznecovpasa807-ui/Blits-Amnezia.WG/main/install.sh)
```

### 🤖 Автоматическая (тихая) установка
Для автоматического развертывания без диалоговых окон используйте аргументы командной строки:

```bash
# Пример бесшумной установки на 80 порт с кастомным паролем
bash <(curl -fsSL https://raw.githubusercontent.com/kuznecovpasa807-ui/Blits-Amnezia.WG/main/install.sh) -y --port 80 --password "MySuperSecurePassword123!"
```

| :--- | :---: | :---: | :---: | :--- |

| **Amnezia 1 / Legacy** | `awg_legacy` | `43913 / UDP` | AmneziaWG (Lite) | Без параметров `S3/S4`. Предотвращает ошибки парсинга конфигов на старых версиях клиентов. |

## 💻 Консольное меню `blits`

После установки на сервере становится доступна утилита `blits` для быстрого администрирования хоста:

```bash
blits
```

### Возможности CLI-консоли:
* **Мониторинг ресурсов**: Отображает CPU Load, RAM, свободное место, статус контейнеров панели.
* **Менеджер клиентов**: Создание клиентов, блокировка/активация, продление подписок, вывод QR-кода прямо в терминал в виде ASCII-символов.
* **Диагностика**: Тестирование портов на конфликты, просмотр живых логов Docker-контейнеров панели и прокси.
* **Резервные копии**: Создание бэкапов всей базы данных и переменных окружения, быстрое восстановление.
* **Сетевой тюнинг**: Включение BBR Congestion Control в один клик для увеличения скорости скачивания.

## ❓ Часто задаваемые вопросы (FAQ)

**Q: Почему QR-коды меняются каждые 2 секунды?**
*A: Мобильное приложение AmneziaVPN считывает конфигурацию частями, если она слишком большая для одного QR-кода. Слайдер автоматически показывает все части по очереди для удобного сканирования.*

**Q: Что такое Split Tunneling и как им пользоваться?**
*A: Это раздельное туннелирование. Вы можете указать доменные имена (например, `youtube.com`) или сети в панели, и только трафик к этим ресурсам пойдет через VPN, оставляя остальные сайты работать напрямую через вашего провайдера.*

**Q: Можно ли обновиться без потери данных клиентов?**
*A: Да. При использовании `./blits` или ручного обновления база данных и файлы конфигураций в папке `data/` сохраняются и не затираются.*
